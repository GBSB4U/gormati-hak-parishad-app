## परिचय

**गोरमाटी हक परिषद** एक नागरिक-केंद्रित डिजिटल मंच है जो आपको आपके अधिकारों (हक) को प्रस्तुत, रिकॉर्ड और ट्रैक करने में मदद करता है।

### मुख्य सिद्धांत:
- ✅ **पारदर्शी**: आपके मामले की पूरी जानकारी आप देख सकते हैं
- ✅ **गैर-राजनीतिक**: यह किसी भी राजनीतिक पार्टी से संबंधित नहीं है
- ✅ **सुरक्षित**: आपके डेटा की गोपनीयता सुनिश्चित है
- ✅ **नैतिक**: न्याय को आगे बढ़ाने के लिए प्रतिबद्ध

## ऐप के मुख्य भाग

### 1. 🏠 होम पेज (मुख्य पृष्ठ)
यहाँ आप चार मुख्य सेक्शन देखेंगे:

| विकल्प | विवरण |
|--------|--------|
| **📢 सार्वजनिक सूचना** | सभी नागरिकों के लिए आधिकारिक घोषणाएं |
| **⚖️ नागरिक अधिकार** | आपके अधिकार और नीतियां |
| **🏛️ शासन** | परिषद की प्रशासनिक जानकारी |
| **📊 जनगणना फॉर्म** | अपनी समस्या दर्ज करने का तरीका |

### 2. 📊 जनगणना और समस्या फॉर्म (Census & Issue Form)

यहाँ आप अपनी समस्या दर्ज कर सकते हैं:

```
कदम 1: फॉर्म खोलें ("जनगणना / समस्या दर्ज करें" पर क्लिक करें)
कदम 2: अपनी जानकारी भरें (वैकल्पिक - अनिवार्य नहीं)
कदम 3: समस्या का विवरण दें
कदम 4: सबूत जोड़ें (फोटो / ऑडियो)
कदम 5: जमा करें
```

### 3. 📢 सार्वजनिक सूचना (Notice Board)

यहाँ आप सभी सार्वजनिक सूचनाएं देख सकते हैं:
- परिषद की घोषणाएं
- महत्वपूर्ण तिथियां
- पारदर्शी अपडेट

### 4. 🛡️ प्रशासक पैनल (Admin Dashboard)

यह केवल परिषद के संचालकों के लिए है।

## उपयोगकर्ता के लिए निर्देश

### अपनी समस्या दर्ज करना

#### विधि 1: जनगणना फॉर्म के माध्यम से

**चरण 1**: होम पेज पर "📊 जनगणना फॉर्म" पर क्लिक करें

**चरण 2**: फॉर्म में निम्न जानकारी भरें (जो आवश्यक हो):

```
- आपका नाम (वैकल्पिक)
- फोन नंबर (वैकल्पिक)
- गाँव / टांडा का नाम
- समस्या का प्रकार (चयन करें)
- विस्तृत विवरण
- सबूत (फोटो / वीडियो)
```

**चरण 3**: सभी जानकारी सावधानीपूर्वक भरने के बाद "जमा करें" बटन दबाएं

**चरण 4**: आपको एक पुष्टिकरण संदेश मिलेगा

### आपकी समस्या को ट्रैक करना

```
1. होम पेज में लॉग इन करें
2. "मेरी समस्याएं" सेक्शन खोलें
3. अपनी समस्या खोजें
4. निम्न स्थितियों में से एक दिखेगी:
   - ⏳ जमा हुई (Submitted)
   - 🔍 समीक्षा में (Under Review)
   - ✅ समाधान (Resolved)
```

### सार्वजनिक सूचनाएं देखना

```
1. होम पेज पर जाएं
2. "📢 सार्वजनिक सूचना" पर क्लिक करें
3. सभी घोषणाएं देख सकते हैं
4. तिथि के आधार पर नवीनतम सूचनाएं पहले आती हैं
```

## प्रशासक के लिए निर्देश

### प्रशासक पैनल तक पहुँचना

**चरण 1**: `admin.html` पृष्ठ खोलें

**चरण 2**: अपनी प्रशासकीय कुंजी दर्ज करें

**चरण 3**: निम्न विकल्पों में से चुनें:

### प्रशासकीय कार्य

#### 1️⃣ जनगणना डेटा की समीक्षा

```
1. "📊 जनगणना डेटा समीक्षा" पर क्लिक करें
2. सभी जमा की गई समस्याओं की सूची देखें
3. प्रत्येक समस्या की विस्तृत जानकारी देखें
4. समस्या को विभाग में मैप करें
```

#### 2️⃣ सार्वजनिक सूचना प्रकाशित करना

```
कदम 1: "📢 Public Notice जारी करें" पर क्लिक करें
कदम 2: निम्न जानकारी भरें:
   - शीर्षक (Title)
   - तिथि (Date)
   - विषय-वस्तु (Content)

कदम 3: "Generate JSON" बटन दबाएं
कदम 4: JSON डेटा तैयार होगा
कदम 5: इसे डेटा बेस में सहेजें
```

#### 3️⃣ विभाग मैपिंग

```
समस्याओं को सही सरकारी विभाग में भेजना:

1. समस्या की श्रेणी देखें
2. उपयुक्त विभाग चुनें
3. विभाग को सूचित करें
4. प्रगति ट्रैक करें
```
